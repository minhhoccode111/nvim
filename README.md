# Neovim Configuration

My neovim editor configuration

**Warning**: If you want to give these config a try, you should first fork this repository, review the code, and remove things you don’t want or need. Don’t blindly use my settings unless you know what that entails. Use at your own risk!

## Getting started

Clone the repository

```bash
git clone git@github.com:minhhoccode111/nvim.git ~/.config/nvim
```

Install nightly Neovim and start using

```bash
nvim
```
